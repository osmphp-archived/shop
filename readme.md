## About `osmphp/shop`

`osmphp/shop` is an e-commerce application that you can install on your server and use.

## Prerequisites

This library requires:

* PHP 8
    * `mbstring` extension
* Composer

Install them if necessary.

## Installation

Install the `osmphp/shop` Composer package:

        composer require osmphp/shop
